# FlowSchedule

A modern task management and scheduling application built with React and Tailwind CSS.

## 🎨 Features

- ✨ Clean, intuitive UI matching the design mockup exactly
- 📋 Task management with priority and category selection
- 🕐 Time and duration scheduling
- 🎨 Beautiful, responsive design with Tailwind CSS
- 🚀 Fast and lightweight React application
- 📱 Mobile-friendly interface

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/jibranrashid83-prog/FlowSchedule.git
cd FlowSchedule
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## 📦 Build for Production

```bash
npm run build
```

Builds the app for production to the `build` folder.

## 🛠 Technologies Used

- **React 18** - UI library
- **Tailwind CSS** - Modern utility-first CSS framework
- **Lucide Icons** - Beautiful, consistent icon library
